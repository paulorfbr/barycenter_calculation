Requirements Checklist
check_circle Minimum contrast 4.5:1 for normal text, 3:1 for large text/UI components
check_circle All interactive elements keyboard navigable (Tab, Enter, Space, Arrow keys)
check_circle Focus indicators: 2px solid Primary-600, offset 2px — never removed
check_circle Form errors communicated via text + icon, not color alone
check_circle Minimum touch target: 44×44px for all interactive elements (buttons use min-height: 44px)
check_circle Status always paired with icon or label, never color alone
check_circle Table headers properly associated with cells (scope attribute)
check_circle prefers-reduced-motion media query disables all non-essential animations
check_circle Aria labels on icon-only buttons; role/aria-modal on dialogs; aria-live on toasts

8 Motion & Animation

Guiding principle: motion communicates state, not decoration. Keep it fast and purposeful.
Interaction Duration Easing CSS Token
Button hover / press 100ms ease-out --anim-button-hover
Sidebar expand / collapse 200ms ease-in-out --anim-sidebar-toggle

Modal close (scale 1 → 0.95) 140ms ease-in --anim-modal-close
Table row selection 80ms ease-out --anim-table-row
Toast slide in (from right) 220ms ease-out --anim-toast-slide
