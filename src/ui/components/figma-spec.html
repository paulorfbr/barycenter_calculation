5 User Flows & Prototype Connections

Primary User Flow: First-Time Calculation

  1. Dashboard → "Calculate Barycenter" button
  2. Barycenter page (empty state shown)
  3. Select company from dropdown
  4. If company has <2 sites: navigate to Sites via link
  5. Sites page: Add Site (opens modal)
  6. Fill modal → "Add Site" → success toast
  7. Return to Barycenter → run calculation
  8. Result displayed in map + detail card
  9. Approve / Reject result → status updates

Secondary Flow: Company Management

  1. Companies page (list view)
  2. Search / filter → find company
  3. "New Company" button → Create modal opens
  4. Fill form → "Create Company" → success toast
  5. New row appears in table
  6. Click Sites icon (location_on) → Sites filtered by company
  7. Add sites → return to Companies
  8. Click Barycenter icon (my_location) → Barycenter with company pre-selected

8 Implementation Notes for Developers

Angular Component Structure

  • All components are Angular standalone (no NgModules)
  • State management via Angular Signals (signal(), computed())
  • Forms use Reactive Forms (FormBuilder)
  • Modal state: signal<boolean> → conditional @if
  • Page transitions: .page-enter CSS class with @keyframes pageFadeIn
  • Toast notifications via ToastService singleton

Map Integration (Leaflet)

  • Map initializes in ngAfterViewInit via @ViewChild
  • Cleanup in ngOnDestroy via this.map?.remove()
  • Site markers: L.circleMarker with radius proportional to sqrt(weightTons/100)
  • Optimal center: L.divIcon with navy circle + white border
  • Distance lines: dashed L.polyline, primary-400 color, 35% opacity
  • Satellite toggle: swap between OSM and Esri World Imagery tile layers

Accessibility Implementation Checklist

  • Buttons: aria-label on icon-only
  • Modals: role="dialog" aria-modal="true" aria-labelledby="id"
  • Delete modals: role="alertdialog"
  • Tables: aria-label, scope="col"
  • Forms: <label for="id"> association required
  • Errors: role="alert" on error messages
  • Map: role="img" aria-label="…"
  • Activity feed: role="feed" aria-live="polite"
  • Pagination: aria-label="Pagination" role="navigation"
  • Active page: aria-current="page"
  • KPI grid: role="list" aria-label="…"
  • Focus ring: 2px solid primary-600, offset 2px
